About Me

สวัสดีครับผมหมั่นยิน แซม อายุ 17 ปี ปัจจุบันเรียนอยู่ที่โรงเรียนเตรียมอุดมศึกษาครับ ผมเป็นเด็กโควต้าดนตรีความสามารถพิเศษคือการ เป่าฟลุตครับเพราะผมเรียนดนตรีมาตั้งแต่ผมยังเด็กจึงคิดว่าสมัครสอบ เข้าโรงเรียนด้วยดนตรีจะง่ายกว่าแต่จริงๆแล้วส่วนตัวผมมีความสนใจ ด้านเขียนโค้ดโปรแกรมต่างๆมาตั้งแต่ม.ต้นแล้วแต่ว่าพึ่งมา ศึกษาจริงๆตอนขึ้นม.3 ตอนนี้ความสามารถที่มีก็จะเป็น html css python หลักๆที่คิดว่าตัวเองช้ำพอประมาณแล้ว (+java ได้นิดนึง) และก็ได้มีการทำโจทย์พวก leetcode มาบ้างแต่อาจจะไม่ได้เยอะครับตอนทำก็ไม่ค่อย เป็นหรอกครับดูเฉลยเป็นส่วนใหญ่

รางวัลจากที่ต่างๆ ความจริงมีมากกว่านี้แต่เอาไปอันเดียวก่อนครับ

  • ทำเกมยิงจรวจ python

  • สร้างเว็ปไซต์ตึก (กำลังทำ)
